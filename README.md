## 📜 项目概述 (Project Overview)

本项目是基于 **黄金黎明协会 (Hermetic Order of the Golden Dawn)** 历史成员原型的深度心理测评系统。通过 12 道精心设计的灵魂拷问，结合历史考据与现代职场/投资逻辑，为用户精准匹配 8 位魔法大师之一。

### 🔮 八大灵魂原型
* **阿莱斯特·克劳利 (Aleister Crowley)**：意志的极限拓荒者。
* **萨缪尔·马瑟斯 (S. L. MacGregor Mathers)**：体系的底层立法者。
* **艾伦·贝内特 (Allan Bennett)**：极简主义的求道先锋。
* **弗朗西斯·雷加迪 (Israel Regardie)**：逻辑与心理的译者。
* **威廉·巴特勒·叶芝 (W. B. Yeats)**：古典浪漫的灵魂守护者。
* **海伦娜·布拉瓦茨基 (Helena Blavatsky)**：宏大叙事的先知。
* **莫伊娜·马瑟斯 (Moina Mathers)**：视觉艺术与灵性的共振器。
* **弗洛伦斯·法尔 (Florence Farr)**：独立灵魂的表演者。

## ✨ 核心特性 (Features)

* **维多利亚黑金美学**：沉浸式 UI 设计，采用 `Cinzel` 衬线体与动效烟雾转场。
* **硬核计分矩阵**：多维度加分算法，确保性格匹配的准确性。
* **深度内容解析**：单角色报告超 3000 字，涵盖：
    * **性格暗面**（基于《秘密特工666》等史料分析）。
    * **现代职场定位**（针对创业者与精英阶层）。
    * **财富投资建议**（含传统大宗商品与 Web3/加密货币策略）。
    * **先锋穿搭美学**（Rick Owens, Lemaire 等品牌联觉）。
* **海报生成技术**：集成 `html2canvas` 与 `QRCode.js`，支持结果一键保存与社交裂变。

## 🛠️ 技术栈 (Tech Stack)

* **Framework**: [Vue 3](https://vuejs.org/) (Composition API)
* **Styling**: [Tailwind CSS](https://tailwindcss.com/)
* **Icons**: [Lucide Icons](https://lucide.dev/)
* **Generation**: [html2canvas](https://html2canvas.hertzen.com/) & [QRCode.js](https://davidshimjs.github.io/qrcodejs/)
* **Deployment**: GitHub Pages
